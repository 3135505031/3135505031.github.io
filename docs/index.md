# Welcome to HYOI

> 这里是邯郸市第一中学的OIer。

HYOI wiki由所有HYOIer构建，旨在帮助后来的HYOIer更好地学习算法竞赛。

本站欢迎所有选手投稿，包括但不限于OI内容、游记、闲话。

![](images/window.gif)

## OI是什么？

**OI**（Olympiad in Informatics，信息学奥林匹克竞赛）在中国起源于 1984 年，是五大高中学科竞赛之一。

本项目受[OI wiki](https://oi-wiki.org/)启发，在编写过程中参考了诸多资料，在此一并致谢。

CF1917F. Construct Tree

Construct Tree

题面翻译

给你一个数组 \(l_1,l_2,\dots.l_n\) 和一个数字 \(d\)。问你是否能够构造一棵树满足以下条件：

  • 这棵树有 \(n+1\) 个点。
  • 第 \(i\) 条边的长度是 \(l_i\)。
  • 树的直径是 \(d\)。

只需要判断是否有解即可。
\(2\le n\le2000,1\le d\le 2000,1\le l_i\le d\)。
By:Call_me_Eric

题目描述

You are given an array of integers $ l_1, l_2, , l_n $ and an integer $ d $ . Is it possible to construct a tree satisfying the following three conditions?

  • The tree contains $ n + 1 $ nodes.
  • The length of the $ i $ -th edge is equal to $ l_i $ .
  • The (weighted) diameter of the tree is equal to $ d $ .

输入格式

Each test consists of multiple test cases. The first line contains a single integer $ t $ ( $ 1 t $ ) — the number of test cases. The description of the test cases follows.

The first line of each test case contains two integers $ n $ , $ d $ ( $ 2 n , 1 d $ ).

The second line of each test case contains $ n $ integers $ l_1, l_2, , l_n $ ( $ 1 l_i d $ ).

It is guaranteed that the sum of $ n $ over all test cases does not exceed $ 2000 $ .

输出格式

For each test case, output $ $ if it is possible to construct a tree that satisfies all the conditions, and $ $ otherwise.

You can print the letters in any case (upper or lower).

题解

考虑长度最大的那条边，设长度为 \(L\)，那么如果 \(L > \lfloor \frac{d}{2} \rfloor\)，\(L\) 就必然在直径上。

证明考虑若它不在直径上，那么我们总可以将这条边取代直径的某一侧获得更大的直径。

之后，既然它在直径上，那么直径就会成为一条长 \(L\) 的边拼上一条长 \(d-L\) 的路径，若存在另外一条长度为 \(L'\) 的边使得 \(L'>d-L\)，一定无解，证明同上。既然所有边的长度都不超过 \(d-L \le L\)了，那么不在直径上的边总是可以找到一个合法位置的（挂在 \(L\) 与 \(d-L\) 的交界处），于是只需要做一个背包判断剩下的长度能不能拼出 \(d-L\) 即可，时间复杂度 \(\mathcal{O}(nd)\)。

否则 \(L \le \lfloor \frac{d}{2} \rfloor\)，同样考虑这条边 \(L\)，如果它可以在直径上，那么所有其它没有在直径上的边也就可以得到一个合法的位置，判断这个只需要和之前 \(L > \lfloor \frac{d}{2} \rfloor\) 一样，判断剩下的边是否可以拼出 \(d-L\) 即可。

否则它不在直径上，删掉它，设 \(dp_{i,j,k}\) 表示前 \(i\) 条边是否可以拼出一段长为 \(j\) 的路径和一段长为 \(k\) 的路径，值只有 \(0\) 和 \(1\)，可以用std::bitset 优化至 \(\frac{nd^2}{w}\)。之后只需要看是否存在一组 \(j,k\) 满足 \(\min(j,k) \ge L\) 并且 \(j,k\) 可以被拼出即可。

单组数据的时间复杂度为 \(\mathcal{O}\left(\frac{n^2d}{w}\right)\)。
